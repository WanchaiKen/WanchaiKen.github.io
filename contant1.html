CSS SPECIFCITY คือ?

css Specifcity เป็นเทคนิคในการเลือกใช้งาน css selector ที่มีความสำคัญมากที่สุดมาใช้งานกับ Element บนหน้าเว็ปไซต์ โดยแบ่งตามความสำคัญจากมากไปน้อย ดังนี้

1. Id sectector มีความสำคัญมากที่สุดจะถูกนำมาใช้ก่อนเสมอ
2. Class selector จะมีความสำคัญรองจาก Id selector
3. Element selector จะมีความสำคัญน้อยที่สุด
